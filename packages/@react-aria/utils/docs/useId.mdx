## Introduction

`useId` is a utility function that creates an id for components when a default
one is not provided. The id's are autogenerated.

## Example

```tsx
let componentId = useId();
```

The result of the above example will be equivalent to this:

```tsx
let componentId = 'react-aria-2357585012-32';
```
