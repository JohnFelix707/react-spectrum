## Introduction

`filterDOMProps` is a utility function for filtering the properties of an object.
It can be used when passing properties from component declaration to top level
rendered element of that component.

## Example

```tsx
let filtered = filterDOMProps({id: 'react-aria-98', value: 'removeThis'});
```

The result of the above example will be equivalent to this:

```tsx
let filtered = {id: 'react-aria-98'};
```
