## Content

The Content, Footer, Header, and Image components exist as children to other
components providing formatting and semantic meaning as explained in
[Dialog](http://localhost:1234/react-spectrum/Dialog.html#content). It is
important to note that the Header, Content, and Footer content elements
accept any renderable node, not just strings.

### Content

Used to specify the content of a component. Examples of usage
[Dialog](Dialog.html#example) and [IllustratedMessage](IllustratedMessage.html#example).

### Footer

Used to specify a layout region below the content section of a component.
Example of usage [Dialog](Dialog.html#example).

See [the MDN docs](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/footer) for more information.

### Header

Used to specify a layout region providing supplemental title information above the
content section of a component. Example of usage [Dialog](Dialog.html#example).

See [the MDN docs](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/header) for more information.

### Image

Used to add images to components. Example of usage
[Dialog](Dialog.html#examples).

See [the MDN docs](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img) for more information.
