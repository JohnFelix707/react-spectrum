## Content

The Heading, Keyboard, and Text components are
[Typography](https://spectrum.adobe.com/page/typography) related and exist as
children to other components providing formatting, semantic meaning, and hierarchical information.

### Heading

[View guidelines](https://spectrum.adobe.com/page/typography/#Heading)

Used to specify the hierarchy of text, like a title, within a component. Examples of usage
[Dialog](Dialog.html#example) and [IllustratedMessage](IllustratedMessage.html#example).

See [the MDN docs](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/Heading_Elements) for more information.

### Keyboard

Used to format text indicating that is a keyboard command. Example of usage
[Menu](MenuTrigger.html#sematic-elements).

See [the MDN docs](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/kbd) for more information.

### Text

Used to wrap your text elements with other components. Examples of usage
[Dialog](Dialog.html#example) and [Button](Button.html#content).
