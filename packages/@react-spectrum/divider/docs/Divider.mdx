## Frequently encountered issues

### Not in a Flex container

When placing a vertical Divider into a non-flex container, you must specify height attributes on the parent or the Divider will have a height of zero. In addition, all the usual vertical alignment tricks must be used. If you want everything aligned to the top, this is what you'd need to do.
`<span>`'s are necessary around the text so that they can have vertical alignment applied. In general, we recommend using Dividers inside Flex and Grid components.
```tsx example
<div style={{height: '100px'}}>
  <span style={{verticalAlign: 'top'}}>Content left</span>
  <Divider orientation="vertical" marginLeft="10px" marginRight="10px" verticalAlign="top" />
  <span style={{verticalAlign: 'top'}}>Content right</span>
</div>
```
